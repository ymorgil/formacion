Esos casos se listan aparte, en la segunda tabla, para poder revisarlos más adelante si interesa.

## Cómo se ha elaborado esta lista

1. Se abrió la colección `Ingles` de @yeraymorg (207 vídeos) y se recorrió el grid completo mediante scroll, capturando la descripción visible de cada vídeo.
2. Se identificaron las descripciones que citaban explícitamente el nombre de una app, web o herramienta.
3. Para cada recurso nombrado se verificó su existencia y enlace oficial mediante búsqueda web.
4. Los vídeos que anuncian un recurso sin nombrarlo en la descripción se han recopilado aparte, para una posible revisión posterior vídeo a vídeo (incluyendo comentarios, donde muchos creadores sí dejan el nombre o enlace).
