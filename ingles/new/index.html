2. Existential Sentences — There is / There are

La estructura there is / there are sirve para introducir la existencia de algo o alguien en una ubicación o situación. Es impersonal: el sujeto real es el noun phrase que sigue al verbo, y el verbo concuerda con él.

Forma Pronunciación Uso Ejemplo
There is (There's) /ðər ɪz/ Existencia de algo singular o uncountable There's a problem with the system. / There is no water left.
There are /ðər ɑː/ Existencia de algo plural There are three candidates for the job.
There isn't /ðər ˈɪznt/ Negación singular o uncountable There isn't enough time.
There aren't /ðər ɑːnt/ Negación plural There aren't any seats available.
Is there…? /ɪz ðər/ Pregunta singular Is there a pharmacy near here?
Are there…? /ɑː ðər/ Pregunta plural Are there any restrictions?
There was /ðər wɒz/ Existencia en el pasado (singular) There was a long queue outside.
There were /ðər wɜː/ Existencia en el pasado (plural) There were no witnesses.
There has been /ðər həz biːn/ Existencia reciente / Present Perfect singular There has been a significant increase in prices.
There have been /ðər həv biːn/ Existencia reciente / Present Perfect plural There have been several complaints.
There will be /ðər wɪl biː/ Existencia futura There will be a meeting tomorrow.
There might be /ðər maɪt biː/ Posible existencia (modal) There might be delays due to the weather.
There seems to be /ðər siːmz tə biː/ Aparente existencia (formal) There seems to be a misunderstanding.
There tends to be /ðər tɛndz tə biː/ Existencia habitual (formal) There tends to be a lot of traffic at this hour.

⚠️ Errores comunes: - ~~It is a book on the table~~ → ✅ There is a book on the table. - En B2/C1 se espera variedad: usa there seems to be, there tends to be, there has been en lugar de repetir there is. - Cuando la lista incluye elementos mixtos (singular + plural), el verbo concuerda con el primer elemento: There is a pen and two notebooks on the desk.
