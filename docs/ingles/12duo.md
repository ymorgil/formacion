# DUOLINGO

**Let´s** significa `vanos a` y se usa para sugerir hacer algo juntos

---

**Park de car** Hay veces que se pone el verbo sin sijeto en impetarivo para dar ordenes.

---

| Inglés | Pronunciación (aprox. en español) | Significado |
|---|---|---|
| season | SÍ-son | estación |
| eraser | i-RÉI-ser | goma |
| tape | TÉIP | cinta |
| want | UÓNT | querer |
| like | LÁIK | gustar |
| really like | RÍ-li LÁIK | me gusta mucho |
